<p>1）新建管理员密码：<br />
mysqladmin -u root password 新密码<br />
2）修改管理员密码：<br />
mysqladmin -u root -p'旧密码' password 新密码<br />
假设原来的密码是'abc',新的密码是'123456',那么命令是：<br />
mysqladmin -u root -p'abc' password '123456'</p>
<p>3）修改其它用户的密码：<br />
mysqladmin -u 用户名 -p 旧密码 password 新密码</p>
