经典的洗牌问题。

一般来讲洗牌的牌都是固定的。

所以一开始可以顺序存放好各种牌。

然后取总牌总的一半的长度来执行随机交换。

交换掉前面的N/2个牌，基本上这个牌的随机性就已经可以了。

```
var cards = [];

for (var i = 1; i <= 54; i++) { cards.push(i); }     //这个就是顺序保存数据的过程

for(var i = 0; i <= 54 / 2; i ++) { var id1 = parseInt(Math.random(1, 53) * 53), tmp = null; tmp = cards[id1]; cards[id1] = cards[i]; cards[i] = tmp; }</p>

//chrome 执行结果

```
