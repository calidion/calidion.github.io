<p>在命令下面打入：<br />
file filename<br />
就会出来文件的格式。</p>
<p>如：<br />
$ file index.py<br />
index.py: ISO-8859 Java program text</p>
<p>$ file crane.txt<br />
crane.txt: UTF-8 Unicode text</p>
<p>对于GBK的格式文档，结果也是ISO-8859:<br />
file crane.txt<br />
crane.txt: ISO-8859 text</p>
